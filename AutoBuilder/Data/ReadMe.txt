配置文件格式：（文件名以非Define结尾） 
	第一行：备注行，生成程序的代码注释
	第二行：客户端字段名称，如果不填，客户端会忽略此列
	第三行：服务器字段名称
	第四行：数据类型
    第五行：索引标识，填index表示索引，可支持多个索引（第一列默认是主键，主键也可以是索引，但不可以是唯一索引）
	第六行：常量名，填入常量名的列会生成一个常量类。只可以标识一列，且该列必须是string类型。
            该列的每一个值对应常量的名字（会自动把路径拆分。例如：A/B/C，会取C为常量名），主键列（第一列）为常量的值。也可以标识主键列（如果主键列是string类型）
            如果值是int类型，则生成枚举，如果是string类型，则生成const string
	第六行以后，就是一条条的数据

定义文件格式：（文件名以Define结尾） 
	备注列，变量名列，变量类型列，值列
	每一行都是一个配置

忽略文件/文件夹：
	以_开头的文件或文件夹会被忽略掉

允许的数据类型 (数组类型的值用&分隔)
	bool,int,long,float,double,string,
	bool[],int[],long[],float[],double[],string[]
    bool数据：0或不填为false，其他为true
    
文件目录：
	PathConfig.xls(用来配置各种路径)
        源Excel文件（可配置）
        生成的CSV文件（可配置）
        生成的代码文件（可配置）
            Model：存放Model类
            Controller：存放Controller类
            DataManager.cs
